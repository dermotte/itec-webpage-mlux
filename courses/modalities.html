Text documents

For all text documents authors have to ensure they are readable for all participants. For non editable text PDF is preferred, for editable and formatted text either Microsoft Word 97-2003 format or the most recent LibreOffice format. For README files use markdown or plain text.

Emails

For emails I recommend to:

  • provide a meaningful subject line and be short and concise
  • send them in plain text (no HTML, RTF, or PDF with the content)
  • send them with a minimum attachment size (e.g. resize images, use the "reduce size" functions of PDF generators and text editors, etc.)
  • use Dropbox, OneDrive, Google Drive orany other service to send files larger than 5 MB
  • follow the netiquette guidelines, ie. by adding a greeting

Submitting work

If you submit software, text, images or any other work to me make sure to

  • include the name and student id number of every participant
  • you have the license / right to include images, text, audio, source code, or video from 3rd parties (check licenses)
  • re-read before you sent and make sure it's the final version.
